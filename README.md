# credit_risk_classification
3 machine learning models that uses the full breadth of bureau-level credit attributes to classify borrowers by risk turning a 537-dimensional behavioral fingerprint into a single explainable score. 

#Running notebooks on windows
1. Create a virtual environment using
    python -m venv .venv

2. Activate virtual environment
    ./.venv/Scripts/Activate.ps1

3. Install required libraries
    pip install -r requirements.txt
